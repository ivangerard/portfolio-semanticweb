## Portfolio Semantic Web

Build Semantic web using best practice of HTML dan CSS

## Objective
1. Able to use CSS Selector
2. Able to change and use CSS Properties, like color, width and margin

## Technology

1. Gulp
2. HTML
3. CSS

## How to Install
```
git clone
```
```
cd client
```
```
npm install
```
```
gulp serve
```
